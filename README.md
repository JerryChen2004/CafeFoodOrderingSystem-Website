# CafeFoodOrderingSystem
My Cafe Food Ordering Website (Acadamic Work)
<br>
Developed with PHP, HTML/CSS and MySQL
<hr>
<br>
Download the files First
<br>
<br>
Make sure to have install XAMPP 
<br>
1. Stop the MySQL in Services on the device and open XAMPP to start Apache and MySQL 
<br>
2. Click the Admin button on MySQL in XAMPP to access phpMyAdmin
<br>
3. Import the "cafefoodordering.sql" into phpMyAdmin to get the myhotelbooking database with the test data
<br>
4. Click on explorer in XAMPP and insert the "cafefoodordering folder" into "htdocs folder" in the xampp
<br>
Then, you are all set to use the website.
<br>
<br>
The link to access the website in customer side: http://localhost/cafefoodordering/user/index.php
<br>
<br>
The link to access the admin panel of the website: http://localhost/cafefoodordering/admin/index.php
<br>
The login for the admin panel(Admin Name: staff, Password: 12345)
<br>
This also can be found in the admin_cred table in the cafefoodordering database on phpMyAdmin
<br>
<br>
Ensure that there are no other "cafefoodordering" databases in phpMyAdmin and a "cafefoodordering" folder in htdocs folder, except for this one.
